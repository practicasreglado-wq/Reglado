# RegladoBienesRaices

> **Estado: archivado / referencia histórica.** No tiene desarrollo activo.

Proyecto histórico del ecosistema Reglado para inmobiliaria de bienes raíces.

gestionado por equipo externo.

- Landing page corporativa de bienes raíces (Vue 3 + Vite).
