# L7 — Fase 3: RegladoMaps (Cookie HttpOnly)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Aplicar el patrón L7 a RegladoMaps. A diferencia de Energy/Ingenieria, Maps **no tiene `BACKEND/` propio** — hay que crearlo desde cero solo con los endpoints de auth.

**Architecture:** Misma arquitectura que fase 2 (proxy local same-origin → ApiLoging server-to-server con API key). Diferencia: el `BACKEND/` se crea de cero. Maps no requiere CRUD propio (los datos del mapa los sirve `ApiMapa/` aparte) — el nuevo backend es solo proxy de auth.

**Tech Stack:** PHP 8 raw + Apache `.htaccess` para SPA fallback; Vue 3 + Vite. Patrón replicado de Energy/Ingenieria.

**Base dir:** `c:/xampp/htdocs/Reglado/`. Todos los paths son relativos a esta raíz.

**Spec de referencia:** [docs/superpowers/specs/2026-04-30-l7-cookie-httponly-design.md](../specs/2026-04-30-l7-cookie-httponly-design.md)

**Bloqueante:** Fase 1 desplegada y validada (Energy/Ingenieria pueden estar o no — fase 3 es independiente de fase 2).

**Fuera de alcance:** Inmobiliaria_Reglados (fase 4).

Crear `RegladoMaps/BACKEND/bootstrap.php`:

```php
<?php

/**
 * Bootstrap mínimo del BACKEND de Maps.
 *
 * Carga variables del .env. No usa Composer (no hay dependencias PHP), todo
 * con funciones nativas + curl.
 */

// Cargar .env
$envPath = __DIR__ . '/.env';
if (is_file($envPath)) {
    foreach (file($envPath, FILE_IGNORE_NEW_LINES | FILE_SKIP_EMPTY_LINES) as $line) {
        $line = trim($line);
        if ($line === '' || str_starts_with($line, '#')) continue;
        $parts = explode('=', $line, 2);
        if (count($parts) !== 2) continue;
        [$k, $v] = $parts;
        $k = trim($k);
        $v = trim($v);
        // Quitar comillas dobles si las hay
        if ((str_starts_with($v, '"') && str_ends_with($v, '"')) ||
            (str_starts_with($v, "'") && str_ends_with($v, "'"))) {
            $v = substr($v, 1, -1);
        }
        if ($k !== '') {
            putenv($k . '=' . $v);
            $_ENV[$k] = $v;
        }
    }
}

// Headers comunes para responses JSON (sobreescribibles por endpoint)
header('X-Content-Type-Options: nosniff');
header('Referrer-Policy: same-origin');
```

- [ ] **Step 1.3: Crear `auth_cookie.php`**

Crear `RegladoMaps/BACKEND/auth_cookie.php`:

```php
<?php

if (!function_exists('maps_set_auth_cookie')) {
    function maps_set_auth_cookie(string $jwt, int $ttlSeconds = 86400): void
    {
        $appEnv = strtolower((string) (getenv('APP_ENV') ?: 'local'));
        setcookie('auth_token', $jwt, [
            'expires'  => time() + $ttlSeconds,
            'path'     => '/',
            'secure'   => $appEnv !== 'local',
            'httponly' => true,
            'samesite' => 'Lax',
        ]);
    }

    function maps_clear_auth_cookie(): void
    {
        $appEnv = strtolower((string) (getenv('APP_ENV') ?: 'local'));
        setcookie('auth_token', '', [
            'expires'  => time() - 3600,
            'path'     => '/',
            'secure'   => $appEnv !== 'local',
            'httponly' => true,
            'samesite' => 'Lax',
        ]);
    }
}
```

- [ ] **Step 2.1: `sso-exchange.php`**

Idéntico al de Energy (Task A3 de fase 2) cambiando:
- `require_once __DIR__ . '/../bootstrap.php';` (mismo)
- `require_once __DIR__ . '/../auth_cookie.php';` (mismo)
- Llamar a `maps_set_auth_cookie(...)` en lugar de `energy_set_auth_cookie(...)`.

```php
<?php

require_once __DIR__ . '/../bootstrap.php';
require_once __DIR__ . '/../auth_cookie.php';

$code = isset($_GET['code']) ? trim((string) $_GET['code']) : '';
$returnUrl = isset($_GET['return']) ? trim((string) $_GET['return']) : '/';

if ($code === '') {
    http_response_code(400);
    echo "missing code";
    exit;
}

$siteOrigin = (string) (getenv('SITE_ORIGIN') ?: 'http://localhost:5176');
$apiBase    = (string) (getenv('APILOGING_BASE_URL') ?: 'http://localhost:8000');
$apiKey     = (string) (getenv('APILOGING_API_KEY') ?: '');

if ($apiKey === '') {
    error_log('[maps sso-exchange] APILOGING_API_KEY no configurado');
    http_response_code(500);
    echo "service misconfigured";
    exit;
}

$ch = curl_init($apiBase . '/auth/exchange-code');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST           => true,
    CURLOPT_HTTPHEADER     => [
        'Content-Type: application/json',
        'Authorization: Bearer ' . $apiKey,
    ],
    CURLOPT_POSTFIELDS     => json_encode([
        'code'              => $code,
        'requesting_origin' => $siteOrigin,
    ]),
    CURLOPT_TIMEOUT        => 10,
]);
$body = curl_exec($ch);
$httpCode = (int) curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($body === false || $httpCode !== 200) {
    $sep = strpos($returnUrl, '?') === false ? '?' : '&';
    header('Location: ' . $returnUrl . $sep . 'sso_failed=1');
    exit;
}

$data = json_decode($body, true);
if (!is_array($data) || empty($data['token'])) {
    http_response_code(502);
    echo "invalid upstream response";
    exit;
}

maps_set_auth_cookie((string) $data['token']);
header('Location: ' . ($returnUrl !== '' ? $returnUrl : '/'));
exit;
```

```php
require_once __DIR__ . '/../bootstrap.php';

    CURLOPT_TIMEOUT        => 10,
]);
$body = curl_exec($ch);
$httpCode = (int) curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($body === false) {
    http_response_code(502);
    echo json_encode(['error' => 'upstream error']);
    exit;
}

http_response_code($httpCode);
echo $body;
```
